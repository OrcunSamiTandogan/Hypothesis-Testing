# Hypothesis-Testing
This project involves the application of hypothesis testing on different artificial datasets. 
The types of tests are 

1- Normality Tests, 

1.1-Variance Tests, 

2-Correlation Tests, 

3-Stationary Tests, 

4-Parametric Statistical Hypothesis Tests, 

5-Nonparametric Statistical Hypothesis Tests.
